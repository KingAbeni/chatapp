# ChatApp

## Description

ChatApp is a real-time messaging application that enables users to communicate instantly. Built with Node.js on the backend and React on the frontend, it includes core features such as user presence status (online/offline), chat room creation, and real-time message updates.

## Features

- Real-time messaging
- Online/offline user status notification
- Private and public chat room support
- User-friendly interface
- Scalable and modular backend

## Tech Stack

- **Backend:** Node.js, Express, Socket.io
- **Frontend:** React, WebSocket
- **Database:** MongoDB
- **Others:** JWT for authentication
